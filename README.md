# Movie Critics GUI Application

A desktop GUI application developed using Java and SQLite as an academic software project.

## Project Overview

Movie Critics is a desktop application that allows users to browse movies, rate movies, manage watchlists, and track watched movies through a graphical user interface.

The project was developed to practice Java GUI development, database operations, object-oriented programming, and software design documentation.

## Technologies Used

- Java
- SQLite
- SQL
- IntelliJ IDEA
- Visual Paradigm

## Features

- User login system
- Movie browsing interface
- Movie rating functionality
- Watchlist management
- Watched movie logging
- Database operations using SQLite
- GUI-based desktop application
- Object-oriented project structure
- Software design diagrams created with Visual Paradigm

## Project Structure

- `src/` contains the Java source code.
- `Database/` contains the SQLite database file.
- `lib/` contains the SQLite JDBC driver.
- `diagrams/` contains the software design diagram files.

## Database Note

The database file contains sample data for demonstration purposes.
